# Synent Technologies Internship - Task 1: Personal Portfolio Website

A clean, modern, and fully responsive single-page personal portfolio built as part of the Synent Technologies Web Development Internship program.

## 🚀 Features
- **Header**: Clean introduction with name and professional title.
- **About Me**: Brief background information.
- **Skills**: Organized grid display showing current technical competencies.
- **Output / Projects**: Highlights current tasks and works.
- **Contact**: Easy navigation to email, GitHub, and LinkedIn profiles.

## 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox & CSS Grid)
- Cursor / Git

## 📌 Instructions Followed
- Single-page architecture
- Clean layout with proper spacing and alignment
- Proper GitHub repository naming conventions
